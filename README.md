# Projects

Repositorio de projetos desenvolvidos durante o bootcamp de cientista de dados da Alua 2020/2021, sendo :

- Analise de dados do numero de infecções do COVID-19. (Analise de dados)
- Analise de dados do historico de cobertura vacinal do Brasil e comparação com paises vizinhos. (Analise de dados)
- Predição do agravamento do estado de pacientes com COVID-19. (Classificação)

Principais ferramentas : Googlecolab, Python, Pandas, Matplotlib, Sklearn e Boruta.
